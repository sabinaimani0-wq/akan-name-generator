# Akan Naming Website

## Project Name
"Akan Naming Website"

## Author
Imani Sabina

## Project Description
Akan Naming Website is used to calculate a user's birth day from their birthdate and assigns the corresponding Akan name, reflecting a rich Ghanaian cultural tradition where names are linked to the day of birth and carry cultural meaning and identity.

## Project Setup Instructions
To run this project locally:
1. Clone the repository:
2. Open the project folder in VS Code.
3. Open the index.html file in your browser.
No additional installations are needed

## Live Site
View the live website here: https://sabinaimani0-wq.github.io/akan-name-generator/

## Features
* About the Akan Name Generator
* Form to fill in your year of birth and gender
